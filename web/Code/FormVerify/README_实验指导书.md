# 智能表单输入提示与验证系统实验指导书

## 一、实验名称

智能表单输入提示与验证系统——静态网页开发技术与轻量机器学习模型融合实验

## 二、实验流程

### 1. 问题

传统表单验证通常只检查是否为空、格式是否正确，不能判断输入质量。本实验需要实现密码强度判断、风险邮箱识别、实时输入提示和提交拦截。

### 2. 选择模型

采用朴素贝叶斯分类器判断密码强度，采用规则引擎识别临时邮箱和明显弱密码。

### 3. 准备数据

- `data/password_data.csv`：密码强度训练数据集。
- `data/email_data.txt`：临时邮箱域名规则库。

### 4. 训练模型

运行：

```bash
python train/train_password_nb.py
```

训练完成后会生成：

```text
model/password_nb_model.json
```

### 5. 问题求解

前端页面监听输入事件，实时调用模型和规则库，动态显示密码强度、邮箱风险、输入框颜色和提交提示。

### 6. 测试

建议测试：

| 测试项 | 输入样例 | 预期结果 |
|---|---|---|
| 弱密码 | student123 | 提示弱密码 |
| 中等密码 | Power@123 | 提示中等 |
| 强密码 | N7!vQ2#pL9@z | 提示强 |
| 临时邮箱 | abc@mailinator.com | 提示高风险邮箱 |
| 错误邮箱 | abc@@test | 提示格式错误 |
| 两次密码不一致 | 两次输入不同 | 阻止提交 |

## 三、运行步骤

1. 用 VS Code 打开项目文件夹。
2. 运行 `python train/train_password_nb.py`。
3. 右键 `index.html`，选择 `Open with Live Server`。
4. 在浏览器中测试表单。

不要直接双击 `index.html`，否则浏览器可能禁止读取本地 JSON 模型文件。
